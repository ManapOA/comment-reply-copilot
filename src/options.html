Ключи хранятся только в локальном хранилище Chrome на этом компьютере.

Как пользоваться

  1. Открой chrome://extensions.
  2. Включи Developer mode.
  3. Нажми Load unpacked и выбери эту папку extension.
  4. Открой YouTube Studio -> Сообщество -> Комментарии.
  5. Жми Ответить или кнопку AI рядом с комментарием.
