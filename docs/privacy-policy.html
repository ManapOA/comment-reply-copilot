Privacy Policy for TubeReply

Last updated: May 7, 2026

TubeReply is a Chrome extension that helps YouTube Studio creators draft AI-generated replies to visible YouTube Studio comments. The extension reads comment text only when the user requests an AI reply draft.

Data the Extension Processes

The extension may process the following data:

  • YouTube Studio comment text visible on the page.
  • The user's selected AI provider and model.
  • Interface language, reply language, preview language, and tone settings.
  • API keys entered by the user for supported AI providers.
  • Draft reply actions initiated by the user.

How Data Is Used

Comment text is sent to the AI provider selected by the user only when the user requests a reply suggestion. The selected provider returns a draft reply and, when applicable, a preview in the user's selected preview language.

API keys and settings are stored locally in Chrome storage on the user's device. TubeReply does not publish comments automatically.

AI Providers

Depending on the provider selected by the user, comment text and prompt context may be sent to:

  • OpenAI
  • Google Gemini
  • Anthropic Claude
  • Groq
  • OpenRouter

The user must provide their own API key for the selected provider.

Data Storage

TubeReply stores extension settings locally in Chrome storage, including selected provider, model, language settings, tone settings, and API keys. The extension developer does not operate a separate backend server for this version of the extension.

Data Sharing

TubeReply does not sell user data and does not share user data with advertisers or data brokers. Data is sent only to the AI provider selected by the user when the user requests an AI-generated reply draft.

Publishing Comments

TubeReply does not automatically publish YouTube comments. It only inserts suggested replies as drafts so the user can review, edit, and submit them manually.

Limited Use

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Contact

For privacy questions, contact the extension publisher using the contact information listed in the Chrome Web Store developer account.
