TubeReply Privacy Policy

TubeReply reads YouTube Studio comment text only when the user requests an AI reply draft.

Comment text, selected context, and draft/rewrite text may be sent to the AI provider selected by the user, such as OpenAI, Google Gemini, Anthropic, Groq, or OpenRouter.

API keys and settings are stored locally in Chrome storage on the user's device.

The developer does not operate a backend server for this version and does not collect, store, sell, or share user data.

The extension does not publish comments automatically.

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.
